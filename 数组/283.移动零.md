# 283.移动零

![](@attachment/Clipboard_2022-06-05-11-02-46.png)

### 代码
    class Solution {
    public:
        void moveZeroes(vector<int>& nums) {
            int slow = 0, fast = 0;
            int len = nums.size();
            while(fast < len){
                if(nums[fast] != 0){
                    nums[slow] = nums[fast];
                    slow++;
                }
                fast++;
            }
            for( ; slow < len; slow++){
                nums[slow] = 0;
            }
        }
    };

### 思路
> 相当于移除零之后，再将后面元素赋值为零。
